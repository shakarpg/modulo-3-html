Python (linguagem de programação)

Python é uma linguagem de programação de alto nível, multiparadigma e de código aberto, conhecida por sua legibilidade e vasta biblioteca padrão. É amplamente usada em ciência de dados, desenvolvimento web, automação, computação científica, aprendizado de máquina e muito mais.

História

Python foi concebida por Guido van Rossum no final da década de 1980 e lançada publicamente em 1991. Desde então, passou por diversas versões, com a série 3.x introduzindo melhorias de consistência e Unicode. A linguagem é mantida pela Python Software Foundation, com um processo colaborativo de evolução por PEPs (Python Enhancement Proposals).

Características

  • Sintaxe clara e legível, com blocos definidos por indentação.
  • Tipagem dinâmica e forte, com inferência em tempo de execução.
  • Biblioteca padrão abrangente (“batteries included”).
  • Suporte a múltiplos paradigmas (OO, funcional, imperativo).
  • Gestão de pacotes via pip e reposição no PyPI.
  • Portável entre sistemas operacionais (Windows, macOS, Linux).

Sintaxe básica

Abaixo, um exemplo simples que demonstra funções, listas e list comprehension:

# hello.py
def saudacao(nome: str) -> str:
    return f"Olá, {nome}!"

nomes = ["Ana", "Bruno", "Carla"]
mensagens = [saudacao(n) for n in nomes]

for m in mensagens:
    print(m)

Exemplo usando bibliotecas populares para dados:

# dados.py
import pandas as pd

df = pd.DataFrame({"produto": ["A", "B"], "vendas": [10, 23]})
print(df.describe())

Ecossistema

A comunidade Python oferece um ecossistema maduro de bibliotecas e frameworks, como Django e Flask (web), NumPy, Pandas e Matplotlib (ciência de dados), além de FastAPI, Typer e Poetry para produtividade e empacotamento.

Exemplos

Exemplo de API mínima com FastAPI:

# api.py
from fastapi import FastAPI

app = FastAPI()

@app.get("/")
def raiz():
    return {"status": "ok"}
